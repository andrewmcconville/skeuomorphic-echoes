# Specimen Sheet — Metadata Fields

## Per Evolutionary Stage

Each stage in the primary skeuomorph's evolutionary chain is presented as a specimen catalog entry. Not every field is required for every stage — density should feel natural, not forced.

---

## Core Fields (always present)

| Field | Example (Movable Type stage) | Format |
|---|---|---|
| **Specimen name** | Movable Type Composing Stick | Display type, large |
| **Era tag** | Mechanical · c. 1450 | Mono, uppercase label |
| **Date range** | c. 1450 – c. 1880 | Mono |
| **Stage position** | 3 of 6 | Mono, small — locates user in the chain |
| **One-line summary** | Individual letters cast in lead, assembled by hand into lines of text | Body type, 1–2 sentences max |
| **Hero image/illustration** | Centered technical illustration of the artifact | Largest element on the page |

---

## Catalog Fields (select 6–10 per stage)

### Physical properties
- **Medium / Material** — Lead alloy, oak frame, iron chase
- **Dimensions** — 220 × 45 × 30 mm (composing stick)
- **Weight** — Approx. 400g loaded
- **Scale indicator** — 1:1, 1:2, or "detail" (per Zerotype reference)

### Mechanism
- **Input method** — Hand composition: selecting individual sorts from a type case
- **Output** — Inked impression on paper via press
- **Key constraint** — Each letter is a physical object with mass and position; sequence is spatial, not temporal
- **Innovation** — Reusable characters; text no longer requires re-carving per page

### Cultural context
- **Geographic origin** — Mainz, Germany (Gutenberg) / Concurrent: Korea, China
- **Primary users** — Trained compositors in print shops
- **Institutional context** — Guild-regulated craft; 7-year apprenticeship typical
- **Cultural impact** — 1–2 sentences on what this stage changed

### Lineage
- **Preceded by** — ← Hand-lettered manuscript (link to previous stage)
- **Succeeded by** — → Phototypesetting (link to next stage)
- **Skeuomorphic echo** — What from this stage survives in modern interfaces (e.g., "leading" as a CSS property named after lead strips)
- **Ghost term** — A word or phrase still in use that originates here (e.g., "uppercase" / "lowercase" from the physical position of type cases)

### Provenance
- **Collection / Source** — Letterform Archive, San Francisco; digitized 2019
- **Image credit** — Photographer or illustrator attribution
- **Further reading** — 1–2 cited sources (Bringhurst, CHM, etc.)

---

## Optional Fields (for richness, not required)

- **Sound** — Audio description or embedded clip (e.g., "the clack of lead sorts dropping into a composing stick")
- **Smell / Texture** — Sensory note for physicality ("ink-stained wood, metallic tang of fresh type")
- **Patent / Key document** — Reference number or title if applicable
- **Related specimen** — Cross-link to a secondary skeuomorph if relevant
- **Acquisition note** — How/where the source artifact was documented

---

## Layout Principle

Metadata fields are arranged as **key: value pairs in a dense mono-set column**, separated by thin horizontal rules or generous line-height. The density is intentional — it reads like a museum catalog raisonné or a specimen tag pinned beside a display case. The user should feel like they're accessing an archive, not skimming a blog post.

Fields that contain links (preceded by, succeeded by, related specimen) serve as navigation — tapping them moves through the evolutionary chain or cross-links to other exhibits.
