Deliverables

  1. Post a link to your Figma file (use the blue share button and copy link button).
    • This will be the same file/link as your other p2: Skeuomorphic Echoes assignments.
  2. Post one screenshot from Figma showing your Happy Paths & User Tasks (zoom out to capture the entire screen).

 

Requirements

  1. Choose two Happy Paths from your FigJam Sitemap (p2.3: Sitemap & Content Inventory)
    • Reference your sitemap in FigJam, but create your Happy Paths in your Figma Design file on the Research page.
    • Make a separate frame for each Happy Path.
    • Each Happy Path will be a list of actions the user performs on the Happy Path.
    • Formula
      • Action: "Element name" element type (page, text, button, link, menu, image)
    • Example
      1. Open: "Home" page
      2. Click: "Our Menu" button
      3. Click: "McNuggets® & McCrispy® Strips" link
      4. Click: "4 Piece Chicken McNuggets®" link
      5. Scroll: "4 Piece Chicken McNuggets®" page
      6. Click: "Nutrition Summary" accordion menu
      7. Read: "Nutrition Summary" text content
      8. Identify: "Sodium" text content
  2. Write a short User Task for both Happy Paths
    • Make a separate frame for each User Task in your Figma research page.
    • Each User Task will be a script asking a participant to complete a task within the Happy Path.
    • What to Include
      • Context: What is the app, what is the participant about to see?
      • Scenario: What is the participant's current goal or motivation in using the app?
      • Task: What specific task do you want them to accomplish within the app?
    • Example
      1. Context: You'll be using the marketing website for McDonald's
      2. Scenario: You're doing your best to be more health conscious and are curious about McDonald's nutrition.
      3. Task: Using the McDonald's website, find how many milligrams of sodium are in a 4 Piece Chicken McNuggets.
