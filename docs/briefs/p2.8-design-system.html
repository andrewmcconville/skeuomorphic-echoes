Deliverables

  1. Post a link to your Figma file (use the blue share button and copy link button).
    • This will be the same file/link as your other p2: Skeuomorphic Echoes assignments.
  2. Post one screenshot from Figma showing your Design System.

 

Requirements

  1. Colors (Figma Variables)
    1. Create color variables (not color styles) for all colors in the project, for example:
      • Primary
      • Secondary
      • Neutrals
      • Backgrounds
      • Text
    2. On your Research Page, show each color in a design system frame, include:
      • Variable/color name (ex: Primary-500, Primary-700)
      • Hex value (ex: #1E3A8A)
      • Example usage label (ex: buttons, links, copy, backgrounds, active state, disabled state)

  2. Typography (Figma Text Styles)
    1. Create Figma text styles for all text in the project, for example:
      • Heading 1
      • Heading 2
      • Body Copy
      • Label Text
      • Other UI Text: buttons, links, captions, errors, hints, disclaimers
    2. On your Research Page, show each style in a design system frame, include:
      • Style name (Heading 1)
      • Font-family and weight (IBM Plex, Light)
      • Size and line-height (80px/88px)

  3. Components (Figma Components)
    1. Create components for repeated design elements along with variants when applicable, for example:
      • Main header (with menu, info, links) and Slim header (back or close button only)
      • Skeuomorph card (with or without variants)
      • Primary button and secondary button
      • Enabled button and disabled button
      • Default button and hovered button
    2. On your Research Page, show each component in a design system frame, include:
      • Component name: button, link, header, footer, avatar, item
      • An instance of the component or the main component itself

  4. Use the Design System
    1. The design system frame should make use of the design system:
      • Apply your color variables to your design system's swatches
      • Apply your text styles to your design system's text
    2. The design system should be used throughout the hi-fi-prototype
      • Does not have to be used on the Research Page nor the Lo-fi Wireframe Page 

 

Your design system will be applied to your hi-fi prototype starting in p2.9: Hi-fi Home & Primary Skeuomorph, which is due the same week. You might build your design system first, then apply it or you might update your design system as you work on your project.
